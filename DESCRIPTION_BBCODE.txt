[SIZE=4][B]Commands[/B][/SIZE]

[B]Data Collection:[/B]
[LIST]
[*][COLOR=Yellow]/qdc enable[/COLOR] - Enable data collection
[*][COLOR=Yellow]/qdc disable[/COLOR] - Disable data collection
[*][COLOR=Yellow]/qdc status[/COLOR] - Check if collection is enabled
[*][COLOR=Yellow]/qdc show[/COLOR] - Show all tracked quests
[*][COLOR=Yellow]/qdc export[/COLOR] - Export data for GitHub submission
[*][COLOR=Yellow]/qdc export <questId>[/COLOR] - Export specific quest
[*][COLOR=Yellow]/qdc clear[/COLOR] - Clear all collected data
[*][COLOR=Yellow]/qdc messages[/COLOR] - Toggle chat messages on/off
[/LIST]

[B]General:[/B]
[LIST]
[*][COLOR=Yellow]/questie[/COLOR] - Open options menu
[*][COLOR=Yellow]/questie toggle[/COLOR] - Enable/disable Questie
[*][COLOR=Yellow]/questie reload[/COLOR] - Soft reload Questie
[*][COLOR=Yellow]/questie journey[/COLOR] - Open Journey window
[/LIST]
